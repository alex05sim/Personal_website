Drop travel photos in this folder. The World page cards pick them up
automatically — no code changes needed.

Expected filenames (defined in src/lib/portfolio-data.ts):

  berkeley.jpg     -> Berkeley, California
  olney.jpg        -> Olney, Maryland
  fort-meade.jpg   -> Fort Meade, Maryland

Landscape photos work best (the cards crop to 4:3). Keep each under
~500 KB for fast loads - 1600px wide is plenty.
